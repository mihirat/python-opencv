# python-opencv

## setup
```
$ brew install pyenv
$ pyenv install 3.7.0
$ brew install opencv
$ ln -s /usr/local/Cellar/opencv/3.4.2/lib/python3.7/site-packages/cv2.cpython-37m-darwin.so  ~/.pyenv/versions/3.7.0/lib/python3.7/site-packages/

$ pip install -r requirements.txt
```
## run
```
$ python main.py
```
